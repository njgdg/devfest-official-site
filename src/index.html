技术大牛的纯干货分享

为期1天，high到爆的技术沙龙，在这里你可以获取 最新的技术干货，听业内大牛分享自己的知识、经验 和见解，你的身旁将会涌现出很多志同道合的朋友，大家 一起玩技术，聊技术，一起high~！

  • 专场分享

    专注于各个领域的技术专家带来的专题分享，你可以获取时下最新潮的技术。

  • 开放式讨论

    找到与你有着相同技术方向的人进行讨论，或者你需要一对一的经验指导等。

  • Code Lab

    拟定一个有趣的主题，进行现场即兴编程，将你的代码和其他朋友的进行交换评价。

  • 愉快的茶歇

    学累了，玩累了，没关系，我们还为大家准备了丰富可口的餐饮作为茶歇。免费的！
